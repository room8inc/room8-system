## 🔴 今すぐやること（優先）

### 1. 404エラーの修正 ⚠️ **最優先**

**Vercelダッシュボードで確認：**

1. **Settings** → **General** を開く
2. 以下の設定を確認：
   - **Root Directory**: `./` または空白
   - **Framework Preset**: `Next.js`
   - **Build Command**: 空白（自動検出）または `npm run build`
3. 必要に応じて修正して**Save**
4. **再デプロイを実行**
5. サイトが正常に表示されるか確認

**詳細**: [fix-404-error.md](./fix-404-error.md) を参照

---

### 2. Vercelの環境変数設定確認 ⚠️ **重要**

**Vercelダッシュボードで確認：**

1. **Settings** → **Environment Variables** を開く
2. 以下の環境変数が設定されているか確認：
   - [ ] `NEXT_PUBLIC_SUPABASE_URL`
   - [ ] `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - [ ] `SUPABASE_SERVICE_ROLE_KEY`
3. 未設定の場合は追加
4. **再デプロイを実行**

**詳細**: [vercel-env-setup.md](./vercel-env-setup.md) を参照
